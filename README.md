# Rnotes
A web based application to add your notes that uses your local storage of browser to store your notes.


# Features

1. Add Notes.
2. Delete Notes.
3. Search Notes.


# TechStack used.

1. HTML5(Hyper Text Markup Language-5).
2. CSS3 (Cascading Style Sheet-3).
3. BOOTSTRAP(Framework).
4. JavaScript.
